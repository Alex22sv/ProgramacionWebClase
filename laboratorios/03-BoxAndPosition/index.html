Modelo de caja.

Cada elemento HTML que existe dentro de una página web se encuentra dentro de una caja rectangular, la cual posee diversas propiedades y atributos que la constituyen.

Contenido:
En él se encuenta todos los elementos procesados que han sido declarado dentro del mismo. Por ejemplo: Texto, imágenes, otras etiquetas.
Margenes Internos:
O padding; es el espacio que se encuentra entre el contenido y el borde de la caja.
Bordes:
Es el espacio que delimita el contenido interno del elemento; puede considerarse como la frontera del elemento.
Margenes Externos:
O "margin"; es el espacio que separa la caja de un elemento con la de los demas; esta propiedad ya no es considerada en el tamaño de la caja.

Cálculo del tamaño de una caja.

Cada elemento posee una propiedad llamada "box-sizing", la cual determina la forma en que el tamaño de la caja es calculado. Estos valores son calculados a partir del valor asignado a las propiedades del Ancho y Alto, "width" y "height" respectivamente. Por defecto se establece el valor de content-box. A continuación se presentan los principales valores que esta propiedad puede tomar.

Content-box

Establece el valor del "width" y "height" al contenido; los valores tanto del padding como del borde se agragan al tamaño del contenido, formando un tamaño mayor al establecido en el ancho y alto.

Ejemplo:

                        
    .content1{
        box-sizing: content-box; 

        width: 100px;
        height: 50px;

        padding: 10px 5px;

        border: 5px solid black;
    }

Border-box

Establece el valor del "width" y "height" a toda la caja; los valores tanto del padding como del borde son distribuidos junto al contenido, formando un tamaño igual al establecido en el ancho y alto.

Ejemplo:

                        
    .content1{
        box-sizing: border-box; 

        width: 100px;
        height: 50px;

        padding: 10px 5px;

Posición con Float.

Cuando un elemento se coloca bajo el modelo de posicionamiento float, automáticamente se convierte en una caja flotante, lo que significa que se desplaza al inicio o hasta el final del contenedor padre de manera horizontal. Esta disposición de elementos de forma horizontal es la mas básica dentro de CSS.

Ejemplo:

HTML 

                
    <div id="example1">
        <p>
            <span></span> Lorem ipsum dolor sit amet consectetur adipisicing elit. Facere tenetur deleniti autem? Mollitia, iste reprehenderit provident voluptatibus laudantium quos itaque animi, recusandae doloremque maxime eos quaerat suscipit aut odio accusantium!                        
        </p>
        
    </div>

CSS 

                
    #example1 span{
        width: 60px;
        height: 60px;

        margin: 10px;

        background: blueviolet;

        float: right;
    }
