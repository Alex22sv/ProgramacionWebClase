Introducción

¿Qué es CSS?

CSS son las siglas para Cascading Style Sheets (Hojas de estilo en cascada en español). Es un lenguaje que se utiliza para dar estilo a los elementos de los documentos escritos en lenguaje de marcado (HTML, XML, SVG, etc.). CSS determina la manera en como es modelada la información en pantalla.

¿Cómo funciona?

Se da estilo a los diferentes elementos por medio de una serie de reglas. Estas reglas al aplicarse a un documento, modifican la apariencia visual de los elementos de dicho archivo. Mientras CSS se encarga de la parte del diseño de una página, el HTML define el contenido de esta.

Para vincular un documento CSS con el HTML, es necesario añadir la dirección relativa del CSS por medio de la etiqueta <link> y asignando de valor en el atributo href dicha rut del archivo .css de la siguiente forma: 

                
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">

        <!-- Relación del archivo CSS con el HTML -->
        <link rel="stylesheet" href="style.css">

        <title>Ejemplo prueba</title>
    </head>
    <body>
        <h1>
            Así se vincula un archivo CSS
        </h1>
    </body>
    </html>

Sintaxis básica

Un conjunto de reglas CSS está compuesto por las siguientes partes:

  • Selectores: Es un módulo que identificará a ciertos elementos de nuestra página. Las declaraciones asociadas se aplicarán solo a estos elementos. También se conoce como filtro.
  • Propiedad: Es un aspecto o característica de una etiqueta HTML que se desea modificar. Entre los más comunes se tiene el color de fondo, color de la letra, color de hipervínculos, etc.
  • Valor: Son todas aquellas unidades, colores, identificadores que modifican el comportamento de una propiedad. Por ejemplo la propiedad color, podría tener los valores de red o green. 
                
    selector{
        prop1: valor1;
        prop2: valor2;
        prop3: valor3;
        ...
    }

Ejemplo.

Se quiere aplicar un fondo #C5E1A5 a los h1 del sitio web. La regla CSS sería la siguiente

CSS 

                
    h1{
        background : #C5E1A5;
    }

HTML 

                
    <h1>
        Título llamativo
    </h1>

Título llamativo

Selectores

Selectores simples.

Estos son las estrcturas o módulos de selección más simples que existen en CSS, de los cuales derivan selectores más complejos a partir de la combinación y unión de los mismos.

Tipos de selectores.

  • Selector de tipo: Selecciona elementos básicos de la estructura html como body o header, es dicir etiquetas HTML. Este tipo de regla es de gran utilidad para generalizar estilos y realizar plantillas del sitio.

    Ejemplo: Si se quiere generalizar un margen al body de todo el sitio (como es el caso de la presente guía), la regla CSS haciendo uso de este selector sería de la siguiente forma: 

                            
    body{
        margin-left : 20%;
        margin-right : 20%;
    }

  • Selector de clase: Este selecciona elementos que han sido marcado con un identificador a traves del atributo class. La sintaxis de este selector es anteceder un punto al nombre de la clase que modificará: 

                            
    .class {
        /*Rules*/
    }

    Este tipo de selectores conviene cuando se desea seleccionar un conjunto específico de elementos que cumplen la misma función. Por ejemplo:

    HTML 

                            
    <p>
        Este párrafo no es importante.
    </p>

    <p class="important">
        Este párrafo es importante.
    </p>

    <p>
        Este párrafo no es importante.
    </p>
    
    <p class="important">
        Este párrafo es importante.
    </p>

    CSS 

                            
    .important{
        color: red;
    }

    Este párrafo no es importante.

    Este párrafo es importante.

    Este párrafo no es importante.

    Este párrafo es importante.

  • Selector de id: Estos, a diferencia de los dos selectores anteriores, delimitan sus reglas de estilos a los elementos únicos que han sido definidos con el atributo id. Su sintaxis se basa en anteponer un numeral al identificador del id seleccionado: 

                            
    #id{
        /* RULES */
    }

    Este selector es muy específico, por lo que es priorizado ante cualquier otro. Ejemplo:

    HTML 

                            
    <p>
        Este párrafo no es importante.
    </p>

    <p class="important" id="flawless">
        Este párrafo es perfecto.
    </p>

    <p class="important">
        Este párrafo es importante.
    </p>
    
    <p class="important">
        Este párrafo es importante.
    </p>

    CSS 

                            
    #flawless{
        color: green;
        text-shadow: 2px 2px lightblue;
    }

    Este párrafo no es importante.

Combinación de selectores.

Normalmente no solo se hará uso de selectores simples; el la práctica existen situaciones que requieren de la combinación de dos o mas selectores, por lo que existen los siguientes operadores:

  • Combinador elementos individuales: El aperador , selecciona ambos elementos listados;

    Sintaxis 

                            
    A , B{
        /* Rules */
    }

    Ejemplo 

                            
    <h1>
        Título pricipal
    </h1>

    <h2>
        Subtítulo
    </h2>

    <p>
        Párrafo 1
    </p>
                        
                    
                            
    h1 , p{
        /* Seleccionará el h1 y el p */

        background: #F8BBD0;
    }

    Sintaxis 

                            
    A + B{
        /* Rules */
    }

        /* Rules */
    }
