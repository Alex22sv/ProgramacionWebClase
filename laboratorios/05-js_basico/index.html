JavaScript Básico

Introducción JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, de alto nivel y multi-paradigma, que tiene por objetivo brindar dinamismo a los sitios web; no obstante este también es utilizado en entornos y tecnologías como base de su funcionamiento (ej: Node.js). Este lenguaje está basado en el estandar normativo ECMAScript.

JavaScript, como ya se mencionó, es multi-paradigma, siendo sus pricipales vertientes la programación funcional, programación imperativa y el paradigma orientado a objetos. La sintaxis básica del mismo es similar a Java y C++, con el fin de minimizar su curva de aprendizaje; sentencias como if, try...catch, for, while y switch, funcionan de forma similar como lo hacen en dichos lenguajes.

Las capacidades dinámicas de JavaScript incluyen construcción de objetos en tiempo de ejecución, listas variables de parámetros, variables que pueden contener funciones, creación de scripts dinámicos, introspección de objetos, y recuperación de código fuente [MDN].

Primeros pasos.

Para empezar se debe de importar o definir nuestro script dentro del documento HTML. Para ello existen diversas formas, entre las cuales se pueden listar:

Script embebido en las etiquetas.

Como su nombre indica, este método consiste en definir el script dentro de los atributos controladores de eventos de una etiqueta. Cabe mencionar que esto se debe de evitar, a menos que se quiera invocar una función externa en cierto evento específico.

<span onclick = "alert('Hice click');"> Haz click aqui! </span>

Uso de la etiqueta Script.

Dentro de HTML existe una etiqueta específica para la definición de scripts, la cual puede ser embebida en cualquier punto del documento. Esta se ejecutará en el momento en que es definida.

<script>

  //Código JS aquí.

</script>

Importar script externo.

Generalmente los scripts son definidos en archivos externos (con extención .js). Para aplicarlos, se debe de hacer uso del atributo src de la etiquta script, donde se delimitará la dirección relativa del archivo.

<script src="assets/js/javaScript.js"></script>

Tip.

A este punto ya se han abarcado varias temáticas que requieren el uso de archivos externos (CSS y JS), por lo que es recomendable hacer uso de una organización de directorios y archivos para mantener el orden dentro de nuestro sitio web. Por ejemplo:

  • root
    • index.html
    • assets
      • img
        • imagen_1.jpg
        • imagen_2.jpg
        • imagen_3.jpg
      • css
        • styles.css
        • extras.css
      • js
        • JavaScript.js
        • nav_extra.js
        • player.js
