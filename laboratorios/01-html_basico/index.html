¿Que es HTML?

HTML, que significa Lenguaje de Marcado para Hipertextos, es el elemento de construcción más básico de una página web y se usa para crear y representar una página web. Determina el contenido de la página web por medio del uso de etiquetas.

Estructura HTML Básica

Al inicio de todo documento html debería de haber una indicación del tipo de documento. Esto es para informar al navegador de ciertas reglas para interpretar el código. Para documentos que utilizan la versión HTML5, esto se logra con: <!DOCTYPE html>.

Seguido al tipo de documento se utiliza la etiqueta <html>. Todo lo que pertenezca a la página web es colocado dentro de esta etiqueta. Este contenedor representa la raiz del documento.

Dentro de la etiqueta <html> se coloca el encabezado con la etiqueta <head>, que contiene elementos que no son incluidos en el contenido del documento, pero son escenciales para los buscadores y el navegador. Es dentro de esta etiqueta en donde se coloca la meta-data (información sobre el documento), como es el título de la página.

El título es una indicación del contenido de la página, este ayuda a buscadores y facilita la navegación si se tienen varias pestañas abiertas en el navegador. El título se coloca dentro de la etiqueta <title>, que a su vez está dentro de la etiqueta <head>. Es importante no omitir esta etiqueta, esta es el nombre del documento, el hacerlo daría una mala impresión a los visitantes de la página. ¿Confiaría en una página que no tiene siquiera un título?

Lo último es separar el encabezado del cuerpo del documento. Este último es colocado junto al encabezado con la etiqueta <body>. Es aquí donde se incluye todo el contenido del documento y es mostrado por el navegador.

Atributos

Los atributos son modificadores en HTML, siempre son escritos después de la etiqueta y dentro de los delimitadores < >. Es posible agregar más de un atributo a una etiqueta.

<etiqueta atributo="valor">

Los atributos son utilizados debido a que muchas etiquetas por si solas no contienen suficiente información, con los atributos es posible agregar mucha información extra. Por ejemplo:

<html lang="es">

Etiquetas a utilizar:

<h1>:
Es una etiqueta de encabezado que implementan seis niveles de encabezado del documento, <h1> es el más importante, y <h6>, el menos importante.
<p>:
El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.
<img>:
Representa una imagen en el documento.
<b>:
Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando.
<strong>:
Marca con especial énfasis las partes más importantes de un texto.
<i>:
Muestra el texto marcado con un estilo en cursiva o italica.
<em>:
Es el apropiado para marcar con énfasis las partes importantes de un texto.
