¿Que es HTML?

HTML, que significa Lenguaje de Marcado para Hipertextos (HyperText Markup Language) es el elemento de construcción más básico de una página web y se usa para crear y representar visualmente una página web. Determina el contenido de la página web por medio del uso de etiquetas

Estructura HTML Básica

Etiquetas a utilizar:

<h1>:
Es una etiqueta de encabezado que implementan seis niveles de encabezado del documento, <h1> es el más importante, y <h6>, el menos importante.
<p>:
El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.
<img>:
Representa una imagen en el documento.
<b>:
Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando.
<strong>:
Marca con especial énfasis las partes más importantes de un texto.
<i>:
Muestra el texto marcado con un estilo en cursiva o italica.
<em>:
Es el apropiado para marcar con énfasis las partes importantes de un texto.
