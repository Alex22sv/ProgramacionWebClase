JavaScript Intermedio

Arreglos y objetos.

Colecciones de datos.

Con el fin de dar solución a la problematica de almacentar conjuntos de datos, JavaScript implemento el objeto Array, el cual simplifica la manera de instanciar arreglos, los cuales además de proporcionar un control de datos sencillo y de alto nivel, se comportan como listas dinámicas.

Una característica particular de este tipo de arreglos es que no se encuentran limitados a un solo tipo de dato, particularidad heredada del tipeo bajo que maneja JavaScript.

Declaración.

Existen diversas formas para inicializar un arreglo en JavaScript. Por su utilidad se estudirán dos: Hacer uso de la notación de arreglo estándar de JS y hacer uso del contructor del objeto Array.

El primer método consiste en, literalmente, asignarle un arreglo a una variable. La sintaxis consiste en encerrar a los datos (separados por comas) entre corchetes ([]). A continuación se ejemplifica este tipo de declaración:

let listNumbers = [1,23,4,5,45,87];
let listLetters = ['A','h','e','v','F'];
let listColors = ["red","yellow","white","green"];

El segundo método se basa el hecho que un arreglo es una instacia de la clase Array, por lo que puede hacerse uso del constructor de la misma. A partir de ello se puede declarar arreglos de la siguiente forma:

let listNumbers = new Array(1,23,4,5,45,87);
let listLetters = new Array('A','h','e','v','F');
let listMixed = new Array(1,'c',"red",34);
