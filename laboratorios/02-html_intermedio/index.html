¿Qué es HTML5?

Etiquetas de semántica

Manejo de formularios con HTML5

<input>

Es un campo de entrada donde el usuario puede ingresar datos. Uno de los atributos que posee la etiqueta “input” es “type” que especifica el tipo de input como botón, checkbox entre otros o el tipo de datos a ingresar.

Algunos de valores que se pueden colocar en el atributo type son:

  1. button: es un botón que no posee acción, a menos que sea especificado en el atributo onClick.
    <input type="button" value=”Send”>
  2. checkbox: permite seleccionar una o más opciones.
    <input type="checkbox"> Female<input type="checkbox"> Male
    3. Female Male
  3. date: permite ingresar una fecha con el formato dd/mm/aaaa.
    <input type="date" >
  4. email: solo permite que el formato del contenido ingresado corresponda a un email.
    <input type="email" >
  5. password: oculta visualmente el contenido que se ha ingresado en el input.
    <input type="password" >
  6. radio: solo permite seleccionar una opción.
    <input type="radio">Yellow
    <input type="radio">Red
    <input type="radio">Blue
    7. Yellow Red Blue
  7. submit: envía datos de un formulario al controlador que sea especificado.
    <input type="submit" value="Submit" >
  8. text: permite al usuario ingresar texto plano.
    <input type="text" >
  9. file: es un botón que permite seleccionar un archivo local para ser cargado.
    Seleccione un archivo: <input type="file" >

<select >

Es una lista desplegable con distintos elementos que pueden ser seleccionados.

<option >

En esta etiqueta se coloca el elemento que se contendrá la lista.
<select>
<option>Toyota</option>
<option>Mercedes</option>
<option>Audi</option>
</select>

<textarea >

Permite al usuario ingresar múltiples líneas. Se puede especificar el ancho y el alto de las líneas visibles mediante los atributos cols y rows.
<textarea rows="10" cols="30">Hello world!!.</textarea >

<label>

Define la etiqueta de un input.
<label >Name: </label> <input type="text” >
