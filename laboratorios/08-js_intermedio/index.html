Validar formularios.

Para esta sección se mostrarán varios métodos de validación para formularios, haciendo uso de las herramientas proporcionadas por JavaScript.

Validación de entradas de texto en cada actualización.

La siguiente etiqueta HTML pertenciente a algún formulario servirá de ejemplo para esta sección.

<input id="first-field" type="text" name="fstFld" placeholder="Texto de relleno">

En primer lugar se debe de asignar una función controladora al evento "onInput" del objeto que representa a la etiqueta.

var first-field = document.getElementById("first-field");

first-field.oninput = () => {

  //Code Here

}

A partir de esto se pueden controlar y monitorear los cambios que existan en cualquier etiqueta input. En este ejemplo se verificará si el campo se encuentra vacío, y se alterará el estilo a partir de ello.

var first-field = document.getElementById("first-field");

first-field.oninput = () => {

   var valor = first-field.value;

   if(valor == "" || valor == null){

     first-field.style.borderColor = "red";

   }else{

     first-field.style.borderColor = "green";

   }
}
Ejemplo 1

Este solo es un ejemplo, pero las entradas de texto pueden ser controladas por diversas condiciones, desde una frase o palabra específica hasta el uso de expresiones regulares; además no se limita a modificar solo la parte estética, sino que se pueden llegar a alterar los distitos atributos de las etiquetas y con ellos su funcionamiento.
