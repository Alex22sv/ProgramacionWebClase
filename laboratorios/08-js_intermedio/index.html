JavaScript & CSS

JavaScript cuenta con una gran cantidad de métodos y propiedades útiles para manipular el estilo CSS de una página web. Esto es de gran utilidad ya que permite realizar lo que se conocen como estilos dinámicos, que consisten en mostrar un estilo diferente dependiendo de lo que se necesita mostrar.

Element.style

Devuelve un objeto que representa el atributo style del elemento.

Element.setAttribute()

Método que recive un nombre de un atributo y el valor que se quiere establecer y se lo añade al elemento al que se aplica

Element.className

Obtiene y / o establece el valor del atributo class del elemento especificado

Element.id

Obtiene y / o establece el valor del atributo id del elemento especificado
