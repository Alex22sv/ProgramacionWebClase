Introducción.

¿Qué es HTML?

HTML define el contenido de una página web haciendo uso de etiquetas. Deriba de el lenguaje XML el cual sirve para describir información y HTML sirve para darle formato y presentarla en un navegador.

Estructura Básica de un documento HTML. 

                
    <!DOCTYPE html>
    <html>
        <head>
            <title> Título </title>
        </head>
        <body>
            <!-- Code -->
        </body>
    </html>

Al inicio de todo documento HTML debería de haber una etiqueta que indique el tipo de documento. Esto es para informar al navegador de ciertas reglas para interpretar el código. Para documentos que utilizan la versión HTML5, esto se logra con: <!DOCTYPE html>.

Seguido al tipo de documento se utiliza la etiqueta <html>. Todo lo que pertenezca a la página web es colocado dentro de esta etiqueta. Este contenedor representa la raiz del documento.

Dentro de la etiqueta <html> se coloca el encabezado con la etiqueta <head>, que contiene elementos que no son incluidos en el contenido visual del documento, pero son escenciales para los buscadores y navegadores. Es dentro de esta etiqueta donde se coloca la meta-data (información sobre el documento), como es el título, autor y descripción de la página.

El título es una indicación del contenido de la página, este ayuda a buscadores y facilita la navegación si se tienen varias pestañas abiertas en el navegador. El título se coloca dentro de la etiqueta <title>, que a su vez está dentro de la etiqueta <head>. Es importante no omitir esta etiqueta, ¿Confiaría en una página que no muestre su título?

Lo último para completar la estructura HTML, es agregar el cuerpo del documento. Este último es colocado junto al encabezado con la etiqueta <body>. Es aquí donde se incluye todo el contenido del documento y es mostrado por el navegador.

Etiquetas y atributos.

Una etiqueta es una marca con clase que delimita una bloque en los lenguajes basados en XML. Los atributos son modificadores de la etiquetas, siempre son escritos después de esta y dentro de los delimitadores < >. Es posible agregar más de un atributo a una etiqueta. 

                
    <etiqueta atributo1= "Valor" atributo2= "Valor">

Los atributos son utilizados debido a que muchas etiquetas por si solas no contienen suficiente información, con los atributos es posible agregar mucha información extra. Por ejemplo: 

                
    <html lang="es">

Con el atributo lang se está indicando que el documento está escrito en idioma español. También se utilizan atributos al utilizar imágenes:

Etiquetas de texto.

  1. El elemento <p>: Su uso es bastante frecuente en la mayoría de sitios web y sirve para representar párrafos. Ejemplo: 
                            
    <p>
        Este es un parrafo.
    </p>

    Este es un parrafo.

  2. Los elementos desde <h1> hasta <h6>: Son utilizados para representar encabezados, desde el que tiene más importancia hasta el que tiene menos, respectivamente. 
                            
    <h1>Encabezado 1</h1>
    <h2>Encabezado 2</h2>
    <h3>Encabezado 3</h3>
    <h4>Encabezado 4</h4>
    <h5>Encabezado 5</h5>
    <h6>Encabezado 6</h6>

    Encabezado 1

    Encabezado 2

    Encabezado 3

    Encabezado 4

    Encabezado 5
    Encabezado 6

    Cabe mencionar que tanto los elementos <p> como los elementos <h1> hasta los <h6> poseen un valor semántico, por consiguiente el uso que se les dé cumplir con las reglas de semántica establecidas.

  3. El elemento <a>: sirve para definir un hipervínculo que, por lo general, redirige a otra página en el mismo sitio o a otro completamente diferente. Es importante mencionar que el navegador implementa ciertos diseños a éste elemento, de ahí el cambio de aspecto a diferencia de un elemento <p> convencional. 
                            
    <p>
        Este link redirige a otra <a href="google.com" target="_blank">página</a>
    </p>

    Este link redirige a otra página.

  4. El elemento <q>: Sirve para indicar que el texto contenido en él, representa una cita corta en línea 
    5.                     
    <p>
        Como lo dijo un antiguo filósofo griego:
        <q cite="https://www.imdb.com/title/tt0062622/quotes/qt0396921">
            Dime con quién andas y te diré quién eres
        </q>
    </p>

    Como lo dijo un antiguo filósofo griego: Dime con quién andas y te diré quién eres

Etiquetas de formato.

HTML nos ofrece la posibilidad de dar formato al texto tal como lo hacen aplicaciones como Word o cualquier otro software de edición de texto.

Algunos de los elementos que existen para tal finalidad son las siguientes:

  1. El elemento <b>: Aplica el formato de negrita al texto que esté dentro de esta etiqueta. 
    2.                     
    <p>
        Este texto está en <b>negrita.</b>
    </p>

    Este texto está en negrita.

  2. El elemento <strong>: Al igual que <b>, aplica el formato de negrita al texto que se encuentre encerrado; la diferencia radica en que <strong> tiene significado semántico, cuando <b>, únicamente aplica un efecto visual. 
    3.                     
    <p>
        Este texto está <strong>resaltado.</strong>
    </p>

    Este texto está resaltado.

  3. El elemento <i>: Aplica el formato de cursiva al texto que esté dentro de esta etiqueta. 
    4.                     
    <p>
        Este texto está en <i>cursiva.</i>
    </p>

    Este texto está en cursiva.

  4. El elemento <em>: Al igual que <i>, aplica el formato de cursiva al texto que se encuentre encerrado; la diferencia radica en que <em> tiene significado semántico, cuando <i>, únicamente aplica un efecto visual. 
    5.                     
    <p>
        Este texto está <em>resaltado.</em>
    </p>

    Este texto está resaltado.

  5. El elemento <u>: Aplica el formato de subrrayado al texto que esté dentro de esta etiqueta. 
    6.                     
    <p>
        Este texto está <u>subrrayado.</u>
    </p>

    Este texto está subrrayado.

  6. El elemento <small>: Reduce el tamaño del texto que esté dentro de esta etiqueta. 
    7.                     
    <p>
        Este texto es más <small>pequeño.</small>
    </p>

    Este texto es más pequeño.

  7. El elemento <big>: Aumenta el tamaño del texto que esté dentro de esta etiqueta. 
    8.                     
    <p>
        Este texto es más <big>grande.</big>
    </p>

    Este texto es más grande.

  8. El elemento <sub>: Aplica el formato de subíndice al texto que esté dentro de esta etiqueta. 
    9.                     
    <p>
        Este texto es un <sub>subíndice.</sub>
    </p>

    Este texto es un subíndice.

  9. El elemento <sup>: Aplica el formato de superíndice al texto que esté dentro de esta etiqueta. 
    10.                     
    <p>
        Este texto es un <sup>superíndice.</sub>
    </p>

    Este texto es un superíndice.

  10. El elemento <del>: Aplica el formato de tachado al texto que esté dentro de esta etiqueta. 
    11.                     
    <p>
        Este texto está <del>tachado.</del>
    </p>

    Este texto está tachado.

Etiquetas de listas.

HTML ofrece algunos metodos para especificar listas de información. Todas las listas deben contener uno a más elementos. Existen tres tipos de listas:

  1. Listas Ordenadas.

    Las listas ordenadas son utilizadas cuando se desea enfatizar un orden en la información presentada.Se representan con las etiquetas <ol> </ol>. Cada elemento de la lista se escribe dentro las etiquetas: <li> </li>. Visualmente cada elemento de la lista se enmunera automaticamente. Ejemplo: 

                            
    <ol>
        <li>
            Galletas
        </li>

        <li>
            Lechuga
        </li>

        <li>
            Chocolate
        </li>
    </ol>

    Preview:

    1. Galletas
    2. Lechuga
    3. Chocolate

  2. Listas no ordenadas

    Las listas no ordenadas van dentro de la etiqueta de apertura <ul> y de su cierre </ul>. Cada punto que se quiera añadir a la lista, se hará dentro de la etiqueta de apertura <li> y su cierre </li>. La caracteristica principal es que los elementos de la lista se van marcando por símbolos y su orden no interesa. Ejemplo: 

                            
    <ul>
        <li>
            Mercurio
        </li>

        <li>
            Venus
        </li>

        <li>
            La Tierra
        </li>
    </ul>

    Preview:

    • Mercurio
    • Venus
    • La Tierra

  3. Listas de definiciones

    Las listas de definiciones se conforman por una serie de pares término/definición, en ellas se utilizan las etiquetas <dl>, <dt> y <dd>. A continuación se explicarán cada una de estas:

    La etiqueta <dl> deriva del término Definiton list, e indica que dentro de ella se escribirá una lista de definiciones.

    La etiqueta <dt> deriva del término Definition term y dentro de ella irá el término que sera definido.

    Finalmente, la etiqueta <dd> deriva del término Definition description e indica que dentro de ésta irá la definición. 

                            
    <dl>
        <dt>
            Protón
        <dt>
        <dd>
            Partícula elemental del núcleo del átomo, provisto de carga eléctrica positiva.
        </dd>

        <dt>
            Electrón
        <dt>
        <dd>
            Partícula elemental del núcleo del átomo, provisto de carga eléctrica negativa.
        </dd>
    </dl>

    Preview:

    Protón
    Partícula elemental del núcleo del átomo, provisto de carga eléctrica positiva.
    Electrón
    Partícula elemental del núcleo del átomo, provisto de carga eléctrica negativa.

Etiquetas de tablas.

La primera etiqueta a utilizar es <table> </table>. Esta es la etiqueta que define el comienzo y final de una tabla. 

                
    <table>

        <!-- Contenido de la tabla -->
    
    </table>

En HTML las tablas se crean fila por fila, y posteriormente las celdas, por ende se hace uso de las siguientes etiquetas:

  • <tr> </tr>: define el comienzo y el final de una fila de la tabla.
  • <td> </td>: define el comienzo y el final de una celda de la tabla. 
                
    <table>
        <tr>
            <td>
                1 1
            </td>

            <td>
                1 2
            </td>

            <td>
                1 3
            </td>
        </tr>

        <tr>
            <td>
                2 1
            </td>

            <td>
                2 2
            </td>

            <td>
                2 3
            </td>
        </tr>

        <tr>
            <td>
                3 1
            </td>

            <td>
                3 2
            </td>

            <td>
                3 3
            </td>
        </tr>
    </table>
