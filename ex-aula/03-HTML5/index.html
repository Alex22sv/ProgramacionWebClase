¿Qué es HTML5?

HTML5 es la última versión de html. Esta nueva versión junto con CSS3, define los nuevos estándares del desarrollo web. No solo se limita a incorporar nuevas etiquetas o atributos, si no que muchas características nuevas y proveer una plataforma compleja de desarrollo para aplicaciones web (mediante APIs).

Esta nueva version, no solo está designada para sustituir HTML4, si no que también XHTML 1 y DOM nivel 2. Nos permite una mayor interacción entre las paginas web y el contenido multimedia (video, audio, entre otros), así como menos complejidad a la hora de codificar nuestro diseño básico.

Algunas de las nuevas características de HTML5 serían:

  • Semántica: Nuevas etiquetas semánticas para estructurar los documentos HTML. Permite describir con mayor precisión cual es el contenido de la página web.
  • Conectividad: Permite comunicarse con un servidor de maneras nuevas e innovadoras.
  • Sin conexión y almacenamiento: Permite a las páginas web almacenar datos localmente en el lado del cliente y poder operar sin conexión de una forma más eficiente.
  • Multimedia: Nuevos elementos multimedia. Nos otorga un excelente soporte para utilizar contenido multimedia como lo son audio y video nativamente.
  • Gráficos y efectos 2D/3D: Integración de gráficos vectoriales. Proporciona una amplia gama de nuevas características que se ocupan de los gráficos en la web como lo pueden ser canvas 2D, WebGL, SVG (como sustitución de <object> ), entre otros.
  • Rendimiento e integración: Proporciona una mayor optimización de la velocidad y un mejor uso del hardware.
  • Acceso al dispositivo: Proporciona APIs para el uso de varios componentes internos de entrada y salida de nuestro dispositivo.
  • CSS3: Nos ofrece una nueva gran variedad para hacer diseños más sofisticados.

Semántica

Una de las novedades que se mencionó anteriormente son las etiquetas nuevas que se incorporan en HTML5, las cuales pueden ser utilizadas en nuestras páginas estáticas. Ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores.

Una etiqueta se califica como semántica si tiene que ver con el significado, es decir, si nos informa sobre lo que trata su contenido. Por ejemplo, la etiqueta <section> nos dice que contiene una sección o un capítulo dentro de la página. Frente a las etiquetas semánticas se tiene otros tipos de etiquetas como a las que afectan al formato, como <ul>, a multimedia, como <video>, entre otras.
Estas nuesvas etiquetas se pueden clasificar en dos grupos:

  • Etiquetas que extienden a las actuales, como <video>, <audio> o <canvas>, y que además agragan nuevas funcionalidades a los documentos HTML (que se pueden controlar desde javascript).
  • Etiquetas que componen la web semántica, es decir, que no incluyen nuevas funcionalidades pero sirven para estructurar sitios web, y añadir un significado concreto, más allá de las etiquetas generales com <div>.
